# W3Css.Blazor 0.1.0

Initial development release for the W3.CSS-backed Blazor component library.

## Highlights

- Bundled W3.CSS 5.01 static asset.
- Lightweight Razor component wrappers for common W3.CSS patterns, with one
  canonical component per concept.
- Design-token theming layer (`w3-theme.css` + `W3Theme` / `W3ThemeProvider`):
  brand the whole app from one place, themed-by-default, with dark mode and no
  per-mode app CSS.
- Documentation site with W3.CSS-style topic pages.
- bUnit test coverage for rendering, binding, interaction, and state behavior.
- Release package version pinned to 0.1.0.
