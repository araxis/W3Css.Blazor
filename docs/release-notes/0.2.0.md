# W3Css.Blazor 0.2.0

Release date: 2026-06-01

## Highlights

- Completed the second-round app component expansion and quality pass.
- Added reusable app primitives including action rows, empty states, app shell, app bar, toolbar, data table, date range picker, drawer, bottom navigation, popover, menu, swipe area, and page content navigation.
- Expanded form controls with required-state, validation, ARIA, and input-time behavior polish.
- Added keyboard and accessibility polish across tabs, rating, stepper, pagination, bottom navigation, menu, dropdown, popover, sidebar, modal, message box, drawer, alert, and focus handling.
- Added theme tokens for brand, semantic status, dark mode, and focus rings while keeping the bundled W3.CSS source file pristine.
- Completed component topic API tables, categorized docs coverage, and package consumer smoke tooling for the current app primitives.
