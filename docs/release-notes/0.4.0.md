# W3Css.Blazor 0.4.0

`0.4.0` is a non-breaking quality-hardening release focused on adoption confidence.

## Highlights

- Expanded XML documentation coverage for exported public APIs, component types, component parameters, theme types, service/options types, and event data types.
- Replaced the selected-public-API documentation check with a broader guard that fails when exported public API summaries are missing.
- Added a docs/API parity guard that compares public component parameters with their topic-page API tables.
- Added a docs browser sweep tool for top adoption and high-risk interactive routes, checking page load, console errors, and horizontal overflow at desktop and mobile widths.
- Refreshed release documentation and version metadata for the `0.4.0` package.

## Compatibility

- No breaking API changes.
- No broad new component families.
- No vendored `w3.css` changes; W3.CSS 5.01 remains bundled unchanged.

## Verification

- Build, test, pack, local package smoke, docs browser sweep, and hygiene scans are expected before tagging `v0.4.0`.
