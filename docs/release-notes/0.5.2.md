# W3Css.Blazor 0.5.2

`0.5.2` is a non-breaking navigation patch release. It keeps the public component surface stable and publishes the root-link fix for starter-kit and app-shell navigation.

## Changes

- Fixed `W3NavMenuItem` so `Href=""` renders an app-root anchor instead of behaving like a command button or current-page empty link.
- Added regression coverage for root-link active state when navigating from nested routes.
- Refreshed nav menu docs, README release links, Versions docs, changelog, starter label, package metadata, and memory files.
- Kept the vendored W3.CSS source unchanged.

## Verification

- `dotnet build W3Css.Blazor.slnx --configuration Release /nr:false`
- `dotnet test W3Css.Blazor.slnx --configuration Release --no-build /nr:false`
