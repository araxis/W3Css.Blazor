# W3Css.Blazor 0.5.3

`0.5.3` is a non-breaking toast spacing patch release. It keeps the public component surface stable and publishes the package-level toast layout polish.

## Changes

- Fixed `W3Toast` close-button spacing so the close action participates in the toast grid layout instead of floating over the panel padding.
- Regenerated `_content/W3Css.Blazor/w3css-blazor.css` so package consumers receive the improved toast layout through the single documented stylesheet.
- Added release-quality coverage for the toast grid and padding rules in the bundled package stylesheet.
- Updated README, Versions docs, changelog, starter label, package metadata, and memory files.
- Kept the vendored W3.CSS source unchanged.

- `dotnet build W3Css.Blazor.slnx --configuration Release /nr:false`
- `dotnet test W3Css.Blazor.slnx --configuration Release --no-build /nr:false`
