# W3Css.Blazor 0.7.0

0.7.0 is a non-breaking live gallery release. It adds a top-level Gallery docs route with copyable app-screen and site-section examples built from the existing package surface.

## Highlights

- Added `/gallery` with live Razor previews for operations dashboards, settings forms, searchable data tables, product sections, pricing/features sections, and contact/FAQ content.
- Linked Gallery from docs navigation, search, home adoption actions, the component index, starter guidance, patterns, and README.
- Extended docs smoke and browser sweep coverage so the Gallery route remains part of release verification.

## Verification Commands

```bash
dotnet build W3Css.Blazor.slnx --configuration Release /nr:false
dotnet test W3Css.Blazor.slnx --configuration Release --no-build /nr:false
dotnet pack src/W3Css.Blazor/W3Css.Blazor.csproj --configuration Release --no-build --output artifacts/packages /nr:false
pwsh ./tools/package-consumer-smoke.ps1 -PackageVersion 0.7.0 -PackageSource artifacts/packages
pwsh ./tools/docs-browser-sweep.ps1 -BaseUrl http://localhost:5036 -StartServer
pwsh ./tools/starter-browser-sweep.ps1 -BaseUrl http://localhost:5037 -StartServer
```

## Compatibility

- No breaking API changes.
- No broad new component family.
- No vendored W3.CSS edits.
