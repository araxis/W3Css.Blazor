# W3Css.Blazor 0.8.0

0.8.0 is a non-breaking polish release. It tightens the adoption path across the starter kit, gallery, patterns, theming, and release documentation without adding public APIs or changing the vendored W3.CSS source.

## Highlights

- Updated package metadata, README links, Versions docs, changelog, release notes, and memory files to the 0.8.0 release target.
- Fixed the starter kit release action so the app-bar label and toast use one sample-local version constant.
- Refreshed component and roadmap guidance around the current adoption path: Starter Kit, Gallery, Patterns, Theming, and Versions.
- Extended release-quality and browser-sweep coverage to prevent starter release-label drift.

## Release Verification

```bash
dotnet build W3Css.Blazor.slnx --configuration Release /nr:false
dotnet test W3Css.Blazor.slnx --configuration Release --no-build /nr:false
dotnet pack src/W3Css.Blazor/W3Css.Blazor.csproj --configuration Release --no-build --output artifacts/packages /nr:false
pwsh ./tools/package-consumer-smoke.ps1 -PackageVersion 0.8.0 -PackageSource artifacts/packages
pwsh ./tools/docs-browser-sweep.ps1 -BaseUrl http://localhost:5036 -StartServer
pwsh ./tools/starter-browser-sweep.ps1 -BaseUrl http://localhost:5038 -StartServer
```
