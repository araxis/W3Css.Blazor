# W3Css.Blazor 0.9.0

0.9.0 is a non-breaking robustness and reach release. It broadens the supported
target frameworks, completes overlay and keyboard accessibility, scales the data
grid and pagination, and hardens runtime teardown, toast concurrency, and theme
scoping. The vendored W3.CSS source is unchanged and existing parameters keep
their defaults.

## Highlights

- Multi-targets `net8.0`, `net9.0`, and `net10.0` so LTS apps can install the package; adds SourceLink, symbol packages, a package icon, CI warnings-as-errors, and a bundle-drift gate.
- `W3Modal`, `W3Drawer`, and `W3Popover` trap keyboard focus and lock body scroll while open, behind opt-out `TrapFocus` and `LockScroll` parameters.
- Roving tabindex with focus-follow on `W3Tabs` and `W3Stepper`, full keyboard navigation for `W3TreeView`, and a focus-revealed `role="tooltip"` for `W3Tooltip`.
- `W3DataTable` memoizes its filtered and sorted view and adds an `ItemKey` and selection `Comparer`; `W3Pagination` windows page buttons with `BoundaryCount` and `SiblingCount`.
- JS teardown tolerates circuit disconnect, `W3ToastService` is thread-safe, and `W3ThemeProvider` scopes its tokens per instance so nested providers theme independently.
